# React Sortable

Sortable component for ReactJS. Demo [here](https://mgechev.github.io/react-reorderable/).

# API

## Properties

- `handle` - a selector, which provides the handle element

## Events

- `onDragStart`
- `onDrag`
- `onDrop`
- `onChange`
